This is a full-stack project hosted on Heroku, with a SQL Database accompanying it. Using an express server and ORM for the interaction between the user and DB. It allows the user to add a burger that they will eat, and once they do eat it they can update the status!

Weather Dashboard

The weather dashboard was created to display information of how the weather is for a city. Made using the Open Weather API it dynamically shows the weather information for that city that day, in addition it will also show how the weather for the next 5 days, it also allows you to search the previous cities you've looked up by saving it in the search history.
